# Stavningsändringar

Italienskans stavning tjänar uttalet: när böjningen skulle förstöra ett k/g- eller tj/dj-ljud rättas stavningen. Reglerna bygger på [c/g-uttalsreglerna](/sprak/italienska/grammatik/basics/pronunciation).

## h skjuts in — hårt ljud bevaras

| Var | Exempel |
|---|---|
| Substantiv/adjektiv -ca/-ga → plural | ami**ca** → ami**che**, ri**ga** → ri**ghe** |
| -co/-go (med näst-sista-betoning) | fuo**co** → fuo**chi**, al**go** → (lago →) la**ghi** |
| Verb på -care/-gare före i/e | cer**care** → cer**chi**, cer**cherò**; pa**gare** → pa**ghi** |

## i försvinner — mjukt ljud utan dubbelt i

| Var | Exempel |
|---|---|
| Verb på -ciare/-giare före i/e | man**giare** → man**gi**, man**gerò** |
| Verb på -iare | stu**diare** → stu**di** |
| Substantiv på -cia/-gia (obetonat i) | aran**cia** → aran**ce** |

## Undantaget alla älskar att hata

**-co/-go med betoning på tredje stavelsen från slutet** tar mjukt -ci/-gi:

- *amico → ami**ci*** (men *amica → amiche*!)
- *medico → medi**ci***

Jämförelse med svenskan: vi har liknande k/tj-växlingar historiskt (*bok → böcker* men *rik → rike*), så fenomenet "stavning följer ljudlag" är inte främmande — italienskan är bara mer konsekvent.
