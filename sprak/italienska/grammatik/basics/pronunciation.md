# Uttal

Italiensk stavning är nästan helt ljudenlig — kan du reglerna kan du uttala allt. De flesta regler är enkla för svenskar; utmaningarna är c/g-reglerna och dubbelkonsonanterna.

## Vokaler

Fem rena vokaler, alltid tydligt uttalade — inga svenska "mumlande" reduktioner:

| Vokal | Uttal | Exempel |
|---|---|---|
| a | som sv. a i *hall* | *casa* |
| e | öppet /ɛ/ eller slutet /e/ | *bello*, *sera* |
| i | som sv. i | *vino* |
| o | öppet /ɔ/ eller slutet /o/ | *cosa*, *sole* |
| u | som sv. **o** i *bo* (aldrig sv. u!) | *luna* |

## c och g — nyckelreglerna

Uttalet beror på nästa bokstav:

| Stavning | Uttal | Exempel |
|---|---|---|
| c + a/o/u | /k/ | *casa*, *come* |
| c + e/i | /tʃ/ ("tj") | *cena* ("tjena"!), *città* |
| ch + e/i | /k/ | *che*, *chi* |
| g + a/o/u | /g/ | *gatto* |
| g + e/i | /dʒ/ ("dj") | *gelato*, *giorno* |
| gh + e/i | /g/ | *spaghetti* |

Alltså: **h gör ljudet hårt**, **i/e gör det mjukt**. *ciao* = "tjao", *chiave* = "kjave".

## Specialkombinationer

| Stavning | Uttal | Exempel |
|---|---|---|
| gn | /ɲ/ (som spanskans ñ) | *gnocchi*, *signore* |
| gli | /ʎ/ (ungefär "lj") | *famiglia*, *gli* |
| sc + e/i | /ʃ/ ("sj") | *pesce*, *sci* |
| sch + e/i | /sk/ | *schiavo* |
| z | /ts/ eller /dz/ | *pizza*, *zero* |

## Dubbelkonsonanter — svenskarnas fördel

Dubbla konsonanter uttalas **långa**, och här har svenskar ett försprång: svenskan har också lång konsonant efter kort vokal (*vila/villa*)! Skillnaden bär betydelse:

- *pala* (spade) — *palla* (boll)
- *caro* (kär/dyr) — *carro* (kärra)
- *sono* (jag är) — *sonno* (sömn)

Håll ut konsonanten som i svenskans *villa* så sitter det.

## r

Rullande tungspets-r, som traditionellt svenskt r: *Roma*, *arrivederci*.
