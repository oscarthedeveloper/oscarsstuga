# Konjunktioner

Tyskans konjunktioner delas i två familjer med motsatt grammatik — blandar du ihop dem havererar ordföljden.

## Samordnande: påverkar inte ordföljden

**und, aber, oder, denn, sondern** ("plats noll") — som svenskans *och, men, eller, för, utan*:

- *Ich koche, **und** er deckt den Tisch.*
- *Sie ist müde, **denn** sie hat lange gearbeitet.* (*denn* = "för", verb på andra plats!)

**sondern** efter negation = svenskans "utan": *Das ist kein Wein, **sondern** Saft.* (Det är inte vin, utan juice.)

## Underordnande: verbet sist

**weil, dass, wenn, obwohl ...** — se [Bisatser](/sprak/tyska/grammatik/syntax/subordinate-clauses):

- *Sie ist müde, **weil** sie lange gearbeitet **hat**.*

Klassisk fälla: *denn* och *weil* betyder båda "eftersom/för" men kräver olika ordföljd.

## Konjunktionella adverb: inversion

**deshalb** (därför), **trotzdem** (ändå), **dann** (sedan), **danach** (därefter), **außerdem** (dessutom) står i fundamentet → verbet direkt efter. Exakt som svenskans "därför gick jag hem":

- *Ich war krank. **Deshalb bin** ich zu Hause geblieben.*
- *Es regnete. **Trotzdem gingen** wir spazieren.*

## Tvådelade konjunktioner

| Par | Svenska |
|---|---|
| entweder ... oder | antingen ... eller |
| weder ... noch | varken ... eller |
| sowohl ... als auch | såväl ... som |
| nicht nur ... sondern auch | inte bara ... utan också |
| zwar ... aber | visserligen ... men |
| je ... desto | ju ... desto (*Je mehr ich lerne, desto besser verstehe ich.*) |

*je ... desto* är ordagrant svenskans "ju ... desto"!
