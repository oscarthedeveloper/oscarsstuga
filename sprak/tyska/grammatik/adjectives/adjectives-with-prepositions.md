# Adjektiv med fasta prepositioner

Som verben väljer många adjektiv en fast preposition + kasus:

| Adjektiv + preposition | Kasus | Exempel (svenska) |
|---|---|---|
| stolz auf | A | *Ich bin stolz auf dich.* (stolt **över**) |
| zufrieden mit | D | *Sie ist zufrieden mit dem Ergebnis.* (nöjd **med**) |
| interessiert an | D | *Wir sind an einer Lösung interessiert.* (intresserad **av**) |
| verantwortlich für | A | *Er ist verantwortlich für das Projekt.* (ansvarig **för**) |
| abhängig von | D | *Das ist abhängig vom Wetter.* (beroende **av**) |
| verliebt in | A | *Sie ist verliebt in ihren Kollegen.* (kär **i**) |
| böse auf | A | *Bist du böse auf mich?* (arg **på**) |
| eifersüchtig auf | A | *Er ist eifersüchtig auf seinen Bruder.* (avundsjuk/svartsjuk **på**) |
| begeistert von | D | *Ich bin begeistert von der Idee.* (entusiastisk **över**) |
| enttäuscht von | D | *Wir sind enttäuscht von dem Film.* (besviken **på**) |
| freundlich zu | D | *Sei freundlich zu ihm!* (snäll **mot**) |
| typisch für | A | *Das ist typisch für ihn.* (typiskt **för**) |
| bekannt für | A | *Die Stadt ist bekannt für ihren Dom.* (känd **för**) |
| gewöhnt an | A | *Ich bin an die Kälte gewöhnt.* (van **vid**) |

Notera kontrasterna mot svenskan: *stolt över = stolz **auf***, *besviken på = enttäuscht **von***, *arg på = böse **auf***.

## Med satser: da(r)-sammansättningar igen

- *Ich bin stolz **darauf**, dass du gewonnen hast.* (stolt över **att**...)
- *Sie ist **daran** interessiert, mehr zu lernen.*

## Substantiven ärver prepositionen

Besläktade substantiv behåller oftast samma preposition: *stolz auf → der Stolz auf*, *Angst vor → die Angst vor*, *interessiert an → das Interesse an*.
