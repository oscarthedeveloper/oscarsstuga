# Prepositioner för tid & plats

## Plats: systemet i ett svep

| Mål | till | i/på |
|---|---|---|
| Stad | **a** Roma | **a** Roma |
| Land/region | **in** Italia | **in** Italia |
| Person | **da** Marco | **da** Marco (hos) |
| Öppna platser | **in** piazza, **al** mare | samma |

Fasta vanor att memorera: *a casa* (hemma/hem), *a scuola*, *al cinema*, *in ufficio*, *in banca*, *in montagna*, *al ristorante*.

## Tid

| Uttryck | Italienska |
|---|---|
| klockan tre | **alle** tre |
| på måndag (nästa) | lunedì (ingen preposition!) |
| på måndagar | **il** lunedì (artikeln = "varje") |
| i januari | **a**/**in** gennaio |
| på sommaren | **in**/**d'**estate |
| 2020 | **nel** 2020 |
| om två timmar | **tra** due ore |
| för två år sedan | due anni **fa** |
| sedan/i (pågående) | **da** due anni (+ presens!) |
| i/under (avslutad tid) | **per** due ore |

## Tre klassiska fällor

1. "På måndag" = bara *lunedì*; *il lunedì* betyder "på måndagAR".
2. "För ... sedan" = **fa** EFTER tidsuttrycket: *due anni fa*.
3. "Jag har bott här i tre år" = *Abito qui **da** tre anni* (presens + da!).
