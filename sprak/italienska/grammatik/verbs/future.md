# Futurum (futuro semplice)

## Presens duger ofta — som i svenskan

*Domani parto per Roma.* (Imorgon åker jag till Rom.) — precis som svenskan använder italienskan gärna presens om nära framtid.

## Bildning

Infinitiven (utan slut-e; -are blir -er) + ändelserna **-ò, -ai, -à, -emo, -ete, -anno**:

| | parlare | credere | dormire |
|---|---|---|---|
| (io) | parler**ò** | creder**ò** | dormir**ò** |
| (tu) | parler**ai** | creder**ai** | dormir**ai** |
| (lui/lei) | parler**à** | creder**à** | dormir**à** |
| (noi) | parler**emo** | creder**emo** | dormir**emo** |
| (voi) | parler**ete** | creder**ete** | dormir**ete** |
| (loro) | parler**anno** | creder**anno** | dormir**anno** |

## Oregelbundna stammar

| Infinitiv | Futurum | | Infinitiv | Futurum |
|---|---|---|---|---|
| essere | sarò | | andare | andrò |
| avere | avrò | | fare | farò |
| venire | verrò | | potere | potrò |
| volere | vorrò | | dovere | dovrò |
| vedere | vedrò | | sapere | saprò |

## Användning

1. Framtid: *L'anno prossimo studierò a Bologna.*
2. **Antagande om nuet** — mycket vanligt: *Dove sarà Marco? — Sarà a casa.* (Var kan Marco vara? — Han är nog hemma.) Jämför svenskans "han lär vara hemma".
3. Efter *quando/se* om framtid använder italienskan futurum där svenskan har presens: *Quando **arriverò**, ti chiamerò.* (När jag kommer fram ringer jag.)

## Futuro anteriore

*avrò parlato* = "kommer att ha talat"; också för antaganden om det förflutna: *Sarà già partito.* (Han har nog redan åkt.)
