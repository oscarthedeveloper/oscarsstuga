# Konjunktioner

## Samordnande

| Italienska | Svenska |
|---|---|
| e | och |
| ma | men |
| o / oppure | eller |
| però | men/dock (starkare än ma) |
| quindi / perciò | alltså, därför |
| infatti | faktiskt, mycket riktigt |
| anzi | tvärtom, rättare sagt |

- *Vorrei venire, **ma** non posso.* (Jag skulle vilja komma, men jag kan inte.)

*e* blir gärna *ed* före vokal: *ed è* (och är).

## Underordnande med indikativ

| Italienska | Svenska |
|---|---|
| che | att |
| perché | eftersom/därför att |
| quando | när |
| mentre | medan |
| se | om |
| siccome | eftersom (satsinledande) |
| appena | så snart |
| dopo che | efter att |

- ***Siccome** pioveva, sono rimasto a casa.* (Eftersom det regnade stannade jag hemma.)

Notera: *perché* betyder både "varför" (i frågor) och "eftersom" (som svar) — *Perché studi? — Perché mi piace!*

## Underordnande med congiuntivo

Dessa kräver alltid konjunktiv — lär dem som paket:

| Italienska | Svenska |
|---|---|
| benché / sebbene | fastän |
| affinché / perché (avsikt) | för att (syfte) |
| prima che | innan |
| senza che | utan att |
| a meno che non | såvida inte |
| purché | förutsatt att |

- ***Benché sia** tardi, resto ancora un po'.* (Fastän det är sent stannar jag en stund till.)
