# Grundtal

## 0–20

| | | | |
|---|---|---|---|
| 0 null | 6 sechs | 11 elf | 16 sechzehn |
| 1 eins | 7 sieben | 12 zwölf | 17 siebzehn |
| 2 zwei | 8 acht | 13 dreizehn | 18 achtzehn |
| 3 drei | 9 neun | 14 vierzehn | 19 neunzehn |
| 4 vier | 10 zehn | 15 fünfzehn | 20 zwanzig |
| 5 fünf | | | |

Notera *sech**zehn*** och *sieb**zehn*** — stammarna kortas (jfr svenskans *sexton/sjutton* som också avviker!).

## Tiotalen och den berömda omkastningen

| | |
|---|---|
| 20 zwanzig | 60 sechzig |
| 30 dreißig (ß!) | 70 siebzig |
| 40 vierzig | 80 achtzig |
| 50 fünfzig | 90 neunzig |

Tvåsiffriga tal läses **entalet först**: 21 = *einundzwanzig* ("ett-och-tjugo"), 45 = *fünfundvierzig*, 99 = *neunundneunzig*. Detta är den stora omställningen från svenskans *tjugoett* — och allt skrivs som ETT ord:

- 347 = *dreihundertsiebenundvierzig*

Träningstips: telefonnummer och årtal tvingar fram automatiken.

## Hundratal och uppåt

- 100 *(ein)hundert*, 200 *zweihundert*, 1 000 *(ein)tausend*
- 1 000 000 *eine Million* (ett substantiv! plural *Millionen*), 10⁹ *eine Milliarde*, 10¹² *eine Billion*

Här är tyskan och svenskan överens: *Milliarde* = miljard, *Billion* = biljon (till skillnad från engelskans *billion* = miljard!).

## eins eller ein?

- Räkning/fristående: **eins** — *eins, zwei, drei*
- Före substantiv: **ein/eine** med vanliga ändelser — *ein Auto, eine Stunde*
- I sammansättningar: **ein** — *einundzwanzig*

## Decimaler, priser

- Decimal**komma** som svenskan: 3,5 = *drei Komma fünf*
- Priser: 4,99 € = *vier Euro neunundneunzig*
- Årtal läses särskilt: se [Ordningstal & datum](/sprak/tyska/grammatik/numbers/ordinals-dates)
