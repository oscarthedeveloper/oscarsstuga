# Introduktion till prepositioner

Italienskans prepositioner styr inga kasus (pust!) — utmaningen ligger i stället i att **valet av preposition sällan matchar svenskans**, och i att de vanligaste smälter samman med artikeln (*di + il = del*).

## De åtta enkla prepositionerna

| Preposition | Grundbetydelse |
|---|---|
| di | av, om (genitiv) |
| a | till, i, åt |
| da | från, hos, sedan |
| in | i, till |
| con | med |
| su | på, om |
| per | för, genom, till |
| tra/fra | mellan, om (tid) |

- *il libro **di** Maria* (Marias bok)
- *Vado **a** Roma.* (Jag åker till Rom.)
- *Vengo **dalla** Svezia.* (Jag kommer från Sverige.)

## Två saker att lära

1. [**di, a, da**](/sprak/italienska/grammatik/prepositions/di-a-da) — de tre mångsysslarna med flest användningar
2. [**Preposizioni articolate**](/sprak/italienska/grammatik/prepositions/articulated) — sammansmältningarna *del, nel, sul* ... som är obligatoriska

Och som alltid: [verbens fasta prepositioner](/sprak/italienska/grammatik/prepositions/verbs-with-prepositions) måste läras med verbet — *penso **a** te* (jag tänker **på** dig) men *parlo **di** te* (jag pratar **om** dig).
