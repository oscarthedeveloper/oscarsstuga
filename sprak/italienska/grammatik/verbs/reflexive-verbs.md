# Reflexiva verb

Reflexiva verb motsvarar svenskans *sig*-verb: *alzarsi* (resa sig/stiga upp), *chiamarsi* (heta, "kalla sig"), *divertirsi* (roa sig).

## Böjning: pronomen FÖRE verbet

| | alzarsi |
|---|---|
| (io) | **mi** alzo |
| (tu) | **ti** alzi |
| (lui/lei) | **si** alza |
| (noi) | **ci** alziamo |
| (voi) | **vi** alzate |
| (loro) | **si** alzano |

Skillnaden mot svenskan: pronomenet står **före** verbet — *mi alzo* = "jag reser mig" (ordagrant "mig reser").

- *Come ti chiami? — Mi chiamo Oscar.* (Vad heter du? — Jag heter Oscar.)
- *A che ora ti alzi?* (Hur dags stiger du upp?)

## Passato prossimo: alltid essere

Reflexiva verb tar **alltid essere**, och participet kongruerar med subjektet:

- *Maria si è alzat**a** presto.* (Maria steg upp tidigt.)
- *Ci siamo divertit**i**.* (Vi hade roligt.)

## Vanliga reflexiva verb

| Verb | Betydelse |
|---|---|
| chiamarsi | heta |
| alzarsi | stiga upp |
| svegliarsi | vakna |
| lavarsi | tvätta sig |
| vestirsi | klä på sig |
| sentirsi | känna sig, må |
| ricordarsi (di) | komma ihåg |
| sposarsi | gifta sig |

Som i tyskan är några verb reflexiva på italienska men inte på svenska — lär dem som paket.
