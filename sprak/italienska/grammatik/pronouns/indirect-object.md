# Indirekta objektspronomen

Motsvarar svenskans "åt/till mig, dig, honom..." — mottagaren:

| Pronomen | Svenska |
|---|---|
| mi | (till) mig |
| ti | (till) dig |
| **gli** | (till) honom |
| **le** | (till) henne |
| Le | (till) Er |
| ci | (till) oss |
| vi | (till) er |
| gli (loro) | (till) dem |

Bara 3:e person skiljer sig från de direkta: **gli** (honom), **le** (henne). I vardagsspråk används *gli* även för "dem".

- ***Gli** parlo domani.* (Jag pratar med honom imorgon.)
- ***Le** scrivo una mail.* (Jag skriver ett mejl till henne.)
- ***Mi** puoi dare una mano?* (Kan du ge mig ett handtag?)

## Verb som tar indirekt objekt

Ofta med *a* i grundkonstruktionen: *parlare a, scrivere a, telefonare a, dare a, dire a, piacere a*:

- *Telefono **a Maria** → **Le** telefono.*

## piacere — "gillar" baklänges

Svenskans "jag gillar pizza" är på italienska "pizzan behagar mig":

- ***Mi piace** la pizza.* (Jag gillar pizza.)
- ***Mi piacciono** gli spaghetti.* (plural! spagettin behagar mig)
- ***Ti piace** l'Italia?* (Gillar du Italien?)

Verbet kongruerar med det som gillas, inte med den som gillar — samma logik som tyskans *gefallen*.

## Betonade former

Vid emfas eller efter preposition används betonade former: *a me, a te, a lui, a lei, a noi, a voi, a loro*:

- ***A me** piace, **a te** no?* (JAG gillar det — inte du?)
