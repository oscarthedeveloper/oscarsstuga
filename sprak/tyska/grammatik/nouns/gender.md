# Genus

Hur vet man om ett ord är *der*, *die* eller *das*? Ofta avslöjar **ändelsen** genus.

## Pålitliga suffix

| Genus | Suffix | Exempel |
|---|---|---|
| femininum | -heit, -keit | die Freiheit, die Möglichkeit (jfr sv. *-het*) |
| femininum | -ung | die Zeitung, die Wohnung (jfr sv. *-ning*) |
| femininum | -schaft | die Freundschaft (jfr sv. *-skap*) |
| femininum | -ion | die Nation, die Diskussion |
| femininum | -tät | die Universität (jfr sv. *-tet*) |
| femininum | -ik | die Musik, die Politik |
| neutrum | -chen, -lein | das Mädchen, das Büchlein (diminutiv) |
| neutrum | -um | das Museum, das Zentrum |
| neutrum | -ment | das Dokument |
| maskulinum | -ling | der Frühling, der Lehrling |
| maskulinum | -ismus | der Tourismus (jfr sv. *-ism*) |

Praktiskt för svenskar: orden på *-het/-heit*, *-ning/-ung*, *-skap/-schaft*, *-tet/-tät* känns igen direkt — och är alltid femininum.

## Starka tendenser

| Genus | Mönster | Exempel | Undantag |
|---|---|---|---|
| maskulinum | -er (personer/apparater) | der Lehrer, der Computer | die Butter, das Fenster |
| femininum | -e | die Lampe, die Straße | der Name, das Ende |
| neutrum | substantiverade infinitiver | das Essen, das Leben | — |
| neutrum | Ge-kollektiv | das Gebirge, das Gemüse | der Gedanke |

## Betydelsegrupper

- **Maskulinum**: veckodagar, månader, årstider (*der Montag, der Juli, der Winter*); väder (*der Regen, der Schnee*); alkoholdrycker (*der Wein* — men *das Bier*); bilmärken (*der BMW*)
- **Femininum**: motorcykelmärken (*die BMW*), många floder (*die Donau, die Elbe* — men *der Rhein*), siffror som substantiv (*die Drei*)
- **Neutrum**: unga varelser (*das Baby, das Kind*), bokstäver (*das A*), metaller (*das Gold*), hotell och kaféer (*das Ritz*)

## Sammansättningar

Sammansatta ord tar genus av **sista** leden — precis som svenskan (*en stadsplan* för att *plan* är ett en-ord):

- die Stadt + der Plan = **der** Stadtplan
- der Tisch + die Lampe = **die** Tischlampe
