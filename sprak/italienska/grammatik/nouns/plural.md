# Plural

Italienskan bildar plural genom **vokalbyte** — inte ändelse som svenskan:

:::tip Grundschemat

| Singular | Plural | Exempel |
|---|---|---|
| -o | -i | libro → libri |
| -a | -e | casa → case |
| -e | -i | notte → notti, fiore → fiori |

:::

## Stavningsjusteringar

| Typ | Regel | Exempel |
|---|---|---|
| -ca/-ga | → -che/-ghe (k/g-ljudet bevaras) | amica → amiche |
| -co/-go (betonad näst sista) | → -chi/-ghi | fuoco → fuochi |
| -co/-go (övriga) | → -ci/-gi | amico → amici, medico → medici |
| -cia/-gia (obetonat i) | → -ce/-ge | arancia → arance |
| -io (obetonat i) | → -i (ett i) | figlio → figli |
| -io (betonat i) | → -ii | zio → zii |

Ja, *amico → amici* men *amica → amiche* är olika — se [Stavningsändringar](/sprak/italienska/grammatik/sound-changes/spelling-changes).

## Oböjliga i plural

- Ord med slutbetoning: *la città → le città*, *il caffè → i caffè*
- Lånord på konsonant: *il bar → i bar*, *lo sport → gli sport*, *il film → i film*
- Förkortningar: *la foto → le foto*

Artikeln avslöjar numerus: *il film / i film* — som svenskans "ett hus / flera hus" där formen inte ändras.
