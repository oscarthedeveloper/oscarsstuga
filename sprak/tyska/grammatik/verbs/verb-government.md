# Verbens rektion: vilket kasus tar objektet?

Varje tyskt verb "styr" sina objekts kasus. De flesta tar ackusativ, men viktiga grupper avviker — och de måste memoreras, eftersom kasuset är osynligt i den svenska översättningen.

## Verb med dativ

Kärnlistan — alla högfrekventa:

| Verb | Exempel |
|---|---|
| helfen (hjälpa) | *Ich helfe **dir**.* |
| danken (tacka) | *Wir danken **Ihnen**.* |
| gehören (tillhöra) | *Das Buch gehört **mir**.* |
| gefallen (behaga) | *Der Film gefällt **uns**.* (Vi gillar filmen.) |
| schmecken (smaka) | *Die Pizza schmeckt **mir**.* |
| passen (passa) | *Die Jacke passt **dir**.* |
| gratulieren | *Ich gratuliere **dir** zum Geburtstag.* |
| antworten (svara) | *Antworte **mir**!* |
| glauben (tro) | *Ich glaube **ihm** nicht.* |
| folgen (följa) | *Der Hund folgt **dem Mann**.* |
| fehlen (saknas) | *Du fehlst **mir**.* (Jag saknar dig!) |
| gelingen (lyckas) | *Der Kuchen ist **mir** gelungen.* |

Minneshjälp: många är "åt/för någon"-verb — tänk svenskans "hjälpa/tacka/svara **åt någon**" så känns dativen logisk.

## Verb med två objekt (dativ + ackusativ)

Giv- och berättarverben tar dativperson och ackusativsak — som svenskans "ge **någon något**":

- *Ich gebe **dem Kind** (D) **einen Apfel** (A).*
- Vanliga: *geben, schenken, zeigen, erklären, erzählen, bringen, schicken, empfehlen, leihen, wünschen*

## Verb med genitiv

En liten formell grupp — känn igen dem i skrift:

- *gedenken* (hedra minnet av): *Wir gedenken **der Opfer**.*
- *bedürfen* (kräva): *Das bedarf **einer Erklärung**.*
- *sich einer Sache bewusst sein*: *Ich bin mir **des Risikos** bewusst.*

## Två ackusativ

*nennen, kosten, lehren*: *Das kostet **mich** **einen Euro**.* — *Er nannte **ihn** **einen Lügner**.*

## es gibt + ackusativ

Existensuttrycket *es gibt* tar alltid ackusativ: *Es gibt **einen Gott** / **keinen Grund**.*
