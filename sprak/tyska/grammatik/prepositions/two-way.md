# Växelprepositioner

Nio rumsprepositioner — **an, auf, hinter, in, neben, über, unter, vor, zwischen** — tar antingen ackusativ eller dativ (*Wechselpräpositionen*):

:::tip Regeln

**Riktning (vart?) → ackusativ. Befintlighet (var?) → dativ.**

Svenskan gör faktiskt samma distinktion — med *vart/var* och med verbparen *lägga/ligga, ställa/stå, sätta/sitta*!

:::

## Sida vid sida

| Ackusativ (vart?) | Dativ (var?) |
|---|---|
| *Ich gehe **in die** Küche.* (ut i köket) | *Ich bin **in der** Küche.* (i köket) |
| *Er legt das Buch **auf den** Tisch.* (lägger på) | *Das Buch liegt **auf dem** Tisch.* (ligger på) |
| *Wir hängen das Bild **an die** Wand.* | *Das Bild hängt **an der** Wand.* |

## Verbparen — svenskans hemliga vapen

Tyskans par av placerings- och lägesverb motsvarar svenskans exakt:

| Riktning (ack., svaga) | Läge (dat., starka) | Svenska |
|---|---|---|
| legen (legte, gelegt) | liegen (lag, gelegen) | lägga / ligga |
| stellen | stehen (stand, gestanden) | ställa / stå |
| setzen | sitzen (saß, gesessen) | sätta / sitta |
| hängen (hängte) | hängen (hing, gehangen) | hänga (tr.) / hänga (intr.) |

Kan du välja mellan *lägga* och *ligga* på svenska vet du redan vilket kasus tyskan vill ha!

## Betydelser

| Preposition | Betydelse |
|---|---|
| an | vid, på (lodrät yta) |
| auf | på (vågrät yta) |
| hinter | bakom |
| in | i / in i |
| neben | bredvid |
| über | över |
| unter | under; bland |
| vor | framför; före; för ... sedan (dat.) |
| zwischen | mellan |

*vor* som "för ... sedan" tar alltid dativ: *vor zwei Jahren* (för två år sedan).

## Överförd betydelse

Vid abstrakt användning (*warten auf*, *denken an*) är kasus fast och måste läras med verbet — oftast ackusativ:

- *Ich warte **auf den** Bus.* — *Ich denke **an dich**.*
