# di, a & da

De tre flitigaste prepositionerna — värda varsin genomgång.

## di — "av/om" (svenskans genitiv-s)

| Användning | Exempel |
|---|---|
| Ägande | *la casa **di** Marco* (Marcos hus) |
| Material/innehåll | *una maglietta **di** cotone* (en tröja av bomull) |
| Ämne | *parliamo **di** politica* (vi pratar om politik) |
| Ursprung (stad) | *Sono **di** Stoccolma.* (Jag är från Stockholm.) |
| Jämförelse | *più alto **di** me* (längre än jag) |
| Delmängd | *un po' **di** pane* (lite bröd) |

## a — "till/i/åt"

| Användning | Exempel |
|---|---|
| Riktning (stad) | *Vado **a** Roma.* |
| Befintlighet (stad) | *Abito **a** Milano.* (Jag bor i Milano.) |
| Mottagare | *Do il libro **a** Maria.* |
| Klockslag | ***alle** otto* (klockan åtta) |
| Sätt | *fatto **a** mano* (handgjord) |

Notera: städer tar **a** både för "till" och "i" — svenskans skillnad görs inte.

## da — mångsysslaren utan svensk motsvarighet

| Användning | Exempel | Svenska |
|---|---|---|
| Ursprung/från | *Vengo **da** Roma.* | från |
| Hos någon | *Vado **da** Marco.* / *Sono **dal** medico.* | hos/till (person!) |
| Sedan (tid) | *Studio italiano **da** due anni.* | i/sedan (+ presens!) |
| Agent i passiv | *scritto **da** Dante* | av |
| Funktion | *occhiali **da** sole* | solglasögon ("för sol") |
| Värd att | *qualcosa **da** mangiare* | något att äta |

**da + person** är fällan: "Jag går till Marco" = *Vado **da** Marco* (inte ~~a Marco~~). Tänk tyskans *bei/zu* eller franskans *chez*.
