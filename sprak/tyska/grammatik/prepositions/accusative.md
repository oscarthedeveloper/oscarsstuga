# Ackusativprepositioner

Dessa tar alltid **ackusativ**: **durch, für, gegen, ohne, um** (samt *bis* och *entlang*). Rabbla: *durch–für–gegen–ohne–um*!

| Preposition | Betydelse | Exempel |
|---|---|---|
| durch | genom | *Wir gehen durch den Park.* |
| für | för, åt | *Das Geschenk ist für dich.* |
| gegen | mot; vid (tid, ungefärligt) | *Er ist gegen die Wand gefahren.* / *gegen acht Uhr* |
| ohne | utan | *Ich trinke Kaffee ohne Zucker.* |
| um | runt; klockan (exakt tid) | *Wir sitzen um den Tisch.* / *um acht Uhr* |
| bis | till, tills | *bis nächsten Montag* |
| entlang | längs (efter substantivet) | *Wir gehen den Fluss entlang.* |

Tidsminnesregel: **um** acht = klockan åtta (precis), **gegen** acht = vid åttatiden.

## Kontraktioner

*durch das → **durchs***, *für das → **fürs***, *um das → **ums*** (vardagligt).
