# Frågor

## Ja/nej-frågor: bara intonation

Enklare än svenskan — ingen omvänd ordföljd, bara stigande ton (eller frågetecken i skrift):

- *Parli svedese?* (Talar du svenska?)
- *Maria viene stasera?* (Kommer Maria ikväll?)

Svenskans "Talar du...?"-inversion finns alltså inte. Påstående + ? = fråga.

## Frågeord

| Italienska | Svenska |
|---|---|
| chi | vem |
| che (cosa) / cosa | vad |
| dove | var/vart |
| quando | när |
| perché | varför |
| come | hur |
| quanto/a/i/e | hur mycket/många (böjs!) |
| quale/quali | vilken/vilka |

- ***Dove** abiti?* (Var bor du?)
- ***Quanto** costa?* (Vad kostar den?)
- ***Quanti** anni hai?* (Hur gammal är du? — "hur många år har du")

Notera att *quanto* och *quale* böjs: *Quante pizze?* (Hur många pizzor?)

## Sammandragningar

- *dove + è → **dov'è*** (var är): *Dov'è la stazione?*
- *cosa + è → **cos'è*** (vad är)

## Indirekta frågor

Samma ordföljd, inleds med frågeordet eller **se** (om):

- *Non so **dove abita**.* (Jag vet inte var han bor.)
- *Mi chiedo **se viene**.* (Jag undrar om hon kommer.)
