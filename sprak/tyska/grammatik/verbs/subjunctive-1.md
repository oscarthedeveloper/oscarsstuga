# Konjunktiv I (referat)

Konjunktiv I används främst för **indirekt anföring** i nyhetsspråk och formell text. Den signalerar: "detta är vad någon sagt — jag går inte i god för det". Svenskan saknar motsvarighet (vi säger "han påstår att han inte gjort några fel"), så formen måste läras aktivt — framför allt för att **läsa tyska tidningar**.

- Direkt: *Der Minister sagte: „Ich **habe** keine Fehler gemacht.“*
- Referat: *Der Minister sagte, er **habe** keine Fehler gemacht.*

## Bildning

Infinitivstam + **-e-ändelser**:

| | machen | haben | können |
|---|---|---|---|
| ich | mache | habe | könne |
| du | machest | habest | könnest |
| er/sie/es | mach**e** | hab**e** | könn**e** |
| wir | machen | haben | können |
| ihr | machet | habet | könnet |
| sie/Sie | machen | haben | können |

**er/sie/es**-formen är den du ständigt möter i nyhetstext: *er sage, sie komme, er müsse, sie wolle*.

## sein är speciellt

| | sein |
|---|---|
| ich | sei |
| du | seist |
| er/sie/es | sei |
| wir | seien |
| ihr | seiet |
| sie/Sie | seien |

*Die Sprecherin sagte, das Problem **sei** gelöst.*

## När Konjunktiv I ser ut som presens...

...används **Konjunktiv II** i stället (gäller mest pluralformer):

- *Sie sagten, sie **hätten** keine Zeit.*

## Vardagstyska

I vardagligt tal används oftast indikativ eller *würde*: *Er hat gesagt, er kommt später.* — precis som på svenska.
