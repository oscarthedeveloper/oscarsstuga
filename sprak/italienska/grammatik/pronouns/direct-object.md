# Direkta objektspronomen

Motsvarar svenskans *mig, dig, honom, henne, den, det, oss, er, dem* som direkt objekt.

| Pronomen | Svenska |
|---|---|
| mi | mig |
| ti | dig |
| lo | honom/den (m) |
| la | henne/den (f) |
| La | Er (artigt) |
| ci | oss |
| vi | er |
| li | dem (m) |
| le | dem (f) |

## Placering: FÖRE verbet

- ***Ti** amo.* (Jag älskar dig — "dig älskar".)
- ***Lo** vedo domani.* (Jag ser honom imorgon.)
- ***La** compro.* (Jag köper den.)

Svenskans ordföljd ("jag ser **honom**") måste alltså vändas. Undantag: pronomenet hängs **efter** infinitiv, gerundium och (oftast) imperativ:

- *Voglio veder**ti**.* (Jag vill se dig.) — infinitivens -e faller
- *Guarda**mi**!* (Titta på mig!)

## lo/la + vokal → l'

- *L'amo.* (Jag älskar honom/henne.)

## Kongruens i passato prossimo

När *lo/la/li/le* står före passato prossimo böjs participet:

- *Hai visto Maria? — Sì, **l'**ho vist**a**.* (Ja, jag har sett henne.)
- *Dove sono le chiavi? — **Le** ho pers**e**!* (Jag har tappat dem!)

## lo som "det"

*lo* kan syfta på hela påståenden: *Lo so.* (Jag vet [det].) — *Non lo so.* (Jag vet inte.) Två ord du kommer använda dagligen.
