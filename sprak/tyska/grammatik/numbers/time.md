# Klockan

## Att fråga

- *Wie spät ist es?* / *Wie viel Uhr ist es?* — Hur mycket är klockan?
- *Um wie viel Uhr...?* — Hur dags...?

## Officiell tid (24-timmars)

Timme + *Uhr* + minuter — tidtabeller, nyheter, avtalade tider:

- 14:30 = *vierzehn Uhr dreißig*
- 9:05 = *neun Uhr fünf*

## Vardaglig tid (12-timmars)

| Klockan | Tyska |
|---|---|
| 7:00 | *sieben (Uhr)* |
| 7:05 | *fünf nach sieben* |
| 7:15 | *Viertel nach sieben* |
| 7:25 | *fünf vor halb acht*! |
| 7:30 | ***halb acht*** |
| 7:35 | *fünf nach halb acht* |
| 7:45 | *Viertel vor acht* |
| 7:55 | *fünf vor acht* |

:::tip halb acht = halv åtta = 7:30

Tyskans *halb* räknar mot kommande timme — **precis som svenskans "halv åtta"**. Här har svenskar en gratisfördel som engelsktalande saknar. Även *fünf vor halb acht* (7:25) fungerar som svenskans "fem i halv åtta"!

:::

Regionalt (öst/syd): *viertel acht* = 7:15, *drei viertel acht* = 7:45 — känns igen från svenskans ålderdomliga "kvart på/i".

## Tidsprepositioner

| | Exempel |
|---|---|
| um (exakt) | *um acht Uhr* — klockan åtta |
| gegen (ungefär) | *gegen acht* — vid åttatiden |
| von ... bis | *von neun bis fünf* |
| ab | *ab acht Uhr* — från och med åtta |
| seit + D | *seit acht Uhr* — sedan åtta |
| in + D | *in einer Stunde* — om en timme (fällan: inte ~~um~~!) |
| vor + D | *vor einer Stunde* — för en timme sedan |

## Delar av dygnet

*morgens, vormittags, mittags, nachmittags, abends, nachts* — *Ich arbeite **morgens**.* (på morgnarna). Igår kväll = *gestern Abend*; imorgon bitti = *morgen früh* (för att undvika *morgen Morgen*!).
