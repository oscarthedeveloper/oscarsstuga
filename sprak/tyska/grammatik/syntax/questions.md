# Frågor

Tyskans frågebildning fungerar som svenskans — inga överraskningar här.

## Ja/nej-frågor: verbet först

- ***Kommst** du mit?* — "Kommer du med?"
- ***Hast** du das Buch gelesen?* — "Har du läst boken?"

## Frågeordsfrågor: frågeord + verb på andra plats

- ***Wo** wohnst du?* — "Var bor du?"
- ***Warum** hast du das gemacht?* — "Varför gjorde du det?"

| Frågeord | Svenska |
|---|---|
| wer / wen / wem / wessen | vem / vem / vem / vems |
| was | vad |
| wo / wohin / woher | var / vart / varifrån |
| wann | när |
| warum / wieso / weshalb | varför |
| wie | hur |
| wie viel(e) | hur mycket/många |
| welcher | vilken |
| was für ein | vad för en |

Notera att tyskan skiljer **wo** (var) och **wohin** (vart) — som korrekt svenska!

## Indirekta frågor

Inbäddade frågor blir **bisatser** — verbet sist (här skiljer sig tyskan från svenskan):

- *Ich weiß nicht, **wo er wohnt**.* — "Jag vet inte var han bor."
- Ja/nej-frågor tar **ob** (om): *Ich frage mich, **ob sie kommt**.* — "Jag undrar om hon kommer."

## oder? och nicht wahr?

Påhängsfrågor: *oder?* eller *nicht wahr?* — svenskans "eller hur?", "va?":

- *Du kommst doch mit, **oder**?* (Du kommer väl med, eller hur?)
