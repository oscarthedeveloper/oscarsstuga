# Alfabetet

Det italienska alfabetet har bara **21 bokstäver**. Fem av våra välbekanta saknas: **j, k, w, x, y** förekommer bara i lånord och namn (*jeans, weekend, taxi*).

| Bokstav | Italienskt namn | | Bokstav | Italienskt namn |
|---|---|---|---|---|
| a | a | | n | enne |
| b | bi | | o | o |
| c | ci ("tji") | | p | pi |
| d | di | | q | cu |
| e | e | | r | erre |
| f | effe | | s | esse |
| g | gi ("dji") | | t | ti |
| h | acca | | u | u |
| i | i | | v | vu |
| l | elle | | z | zeta |

## Accenttecken

Italienskan markerar betoning på slutvokal med accent: **à, è, é, ì, ò, ù**:

- *città* (stad), *caffè* (kaffe), *perché* (varför), *più* (mer)

Grav accent (è) = öppet uttal, akut (é) = slutet. I praktiken behöver du bara skriva accenten där den hör hemma — den är obligatorisk: *e* (och) och *è* (är) är olika ord!

## h är stumt

**h** uttalas aldrig: *ho* (jag har) låter som "å". Dess jobb är att styra uttalet av c och g — se [Uttal](/sprak/italienska/grammatik/basics/pronunciation).
