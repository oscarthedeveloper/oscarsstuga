# Imperativ (imperativo)

## Formerna

| | parlare | credere | dormire | finire |
|---|---|---|---|---|
| (tu) | parl**a**! | cred**i**! | dorm**i**! | fin**isci**! |
| (Lei) | parl**i**! | cred**a**! | dorm**a**! | fin**isca**! |
| (noi) | parl**iamo**! | crediamo! | dormiamo! | finiamo! |
| (voi) | parl**ate**! | credete! | dormite! | finite! |

- *Scusa!* (ursäkta, till du) / *Scusi!* (ursäkta, till Ni) — hör du skillnaden på italienska gator dagligen.
- *Andiamo!* (Nu går vi!) — noi-formen är svenskans "låt oss".

## Negerad tu-imperativ: non + infinitiv!

En riktig överraskning för svenskar:

- *Parla!* (Tala!) men *Non parlare!* (Tala inte!)
- Övriga personer negeras normalt: *Non parli!* (Lei), *Non parlate!* (voi)

## Oregelbundna tu-former

| Verb | Imperativ |
|---|---|
| andare | va' (vai) |
| fare | fa' (fai) |
| dare | da' (dai) |
| stare | sta' (stai) |
| dire | di' |
| essere | sii |
| avere | abbi |

## Pronomen hängs på

Klitiska pronomen fästs **efter** imperativen (utom Lei-formen):

- *Dimmi!* (Säg mig!) — *Aiutami!* (Hjälp mig!) — *Alzati!* (Res dig!)
- Men Lei: *Mi dica!* (Säg mig! — formellt)

Med apostrofformerna dubblas konsonanten: *da' + mi → **dammi*** (ge mig), *fa' + lo → **fallo*** (gör det).
