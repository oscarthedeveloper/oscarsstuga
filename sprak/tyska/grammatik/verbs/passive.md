# Passiv

## Processpassiv med werden

Tyskans huvudpassiv bildas med **werden + particip** och motsvarar svenskans **bli-passiv** (eller s-passiv):

- *Das Haus **wird gebaut**.* (Huset byggs / blir byggt.)
- *Die Briefe **wurden** gestern **verschickt**.* (Breven skickades igår.)

:::warning Viktigt för svenskar

Svenskans smidiga **s-passiv** (*byggs*, *skickades*) saknas helt i tyskan — du måste alltid använda *werden*-konstruktionen.

:::

| Tempus | Exempel |
|---|---|
| Präsens | Das Auto wird repariert. |
| Präteritum | Das Auto wurde repariert. |
| Perfekt | Das Auto ist repariert **worden**. |
| Futur I | Das Auto wird repariert werden. |
| Med modalverb | Das Auto muss repariert werden. |

Observera specialparticipet **worden** (inte *geworden*) i passivens perfekt.

## Tillståndspassiv med sein

**sein + particip** beskriver **resultatet** — som svenskans "är + particip":

- *Das Fenster **ist geöffnet**.* (Fönstret är öppnat.)

Jämför:

- *Die Tür wird geschlossen.* (Dörren stängs — pågår.)
- *Die Tür ist geschlossen.* (Dörren är stängd — tillstånd.)

## Agenten

**von** + dativ för personer, **durch** + ackusativ för medel/orsak — svenskans "av" respektive "genom":

- *Der Roman wurde **von** Kafka geschrieben.*
- *Die Stadt wurde **durch** ein Erdbeben zerstört.*

## Opersonlig passiv

Tyskan kan göra passiv även av verb utan objekt:

- *Hier **wird** viel **gearbeitet**.* (Här jobbas det mycket.) — svenskan har faktiskt samma konstruktion: "det dansas"!

## Dativobjekt förblir dativ

- *Man hilft **dem Mann**.* → ***Dem Mann** wird geholfen.* (Mannen blir hjälpt.)

## bekommen-passiven

Vardagstyskan har en tredje passiv som lyfter **dativobjektet** till subjekt — med *bekommen* (informellt *kriegen*) + particip:

- Aktiv: *Man schenkt **ihm** ein Buch.* → *Er **bekommt** ein Buch **geschenkt**.* (Han får en bok [given].)
- *Ich **habe** den Job **angeboten bekommen**.* (Jag blev erbjuden jobbet.)

Svenskans "han blev erbjuden/tilldelad" motsvaras alltså av en egen tysk konstruktion.

## Alternativ till passiven

Tyskan undviker ofta passiv där svenskan tar s-form:

- **man**: *Man spricht hier Deutsch.* (Här talas tyska.)
- **sich lassen** (= går att): *Das Problem **lässt sich lösen**.* (Problemet går att lösa — jfr svenskans "låter sig lösas".)
- **-bar-adjektiv**: *machbar* (görbar), *essbar* (ätbar), *unübersetzbar* (oöversättbar) — samma mönster som svenskans *-bar*!
